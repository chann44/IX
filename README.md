# IX

Self-hostable multi-language package registry.

## Overview
IX is a unified package registry that lets you store, manage, and distribute packages across multiple ecosystems from a single system.

## Features
- Multi-language support (JS, Python, Go, Rust, more)
- Self-hosted and open
- Versioning and package management
- CI/CD integration ready
- Secure and scalable

## Getting Started
```bash
# clone the repo
git clone https://github.com/chann44/IX.git

# install dependencies
cd IX 
make install

# run locally
make dev
